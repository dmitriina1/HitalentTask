## Запуск проекта
Перейти в корневую папку проекта, где находится `docker-compose.yaml` и выполнить команду `docker-compose up --build`.
Сервер запустится на порту 8080.

API документировано с помощью Swagger. После запуска сервера перейдите по адресу, чтобы увидеть все endpoint-ы:  **http://localhost:8080/docs**
